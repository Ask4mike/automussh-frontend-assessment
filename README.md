## Quick Start

```bash
# Clone repository
git clone https://github.com/yourusername/automussh-landing-page.git

# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
